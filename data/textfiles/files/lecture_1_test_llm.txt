# Лекция 2: Массивы

## О массивах и их характеристиках

Массив - это структура данных, предназначенная для хранения элементов одного типа. Массив может состоять из целых чисел, вещественных чисел, символов или строк. В реальной жизни массивы можно назвать упорядоченный набор инструментов или же набор елочных игрушек.

Массив должен иметь имя (идентификатор). Этот термин может быть сформулирован по-разному, но в большинстве языков программирования индекс массива начинается с нуля.

Основные характеристики массивов:
1. Массивы имеют имя (идентификатор).
2. Элементы в массиве располагаются последовательно, то есть в памяти они занимают ячейку за ячейкой.
3. Массивы имеют конечный размер, поскольку объем памяти у нас всегда ограничен.

## Операции с массивами в C#

Создание массива в C# состоит из трех основных этапов:
1. Указание типа данных для элемента массива.
2. Указание идентификатора или имени массива.
3. Заполнение массива значениями.

Пример создания массива в C#:
```csharp
int[] numbers = new int[5];
numbers[0] = 1;
numbers[1] = 2;
numbers[2] = 3;
numbers[3] = 4;
numbers[4] = 5;
```

В этом примере мы создали массив `numbers`, который состоит из пяти элементов типа `int`. Затем мы заполнили его значениями от 1 до 5.

## Циклы for и foreach в C#

В языке C# существуют два разновидности циклов для работы с массивами: цикл `for` и цикл `foreach`.

Цикл `for` позволяет выполнять блок кода определенное количество раз. Пример использования цикла `for` в C#:
```csharp
int[] numbers = new int[5];
numbers[0] = 1;
numbers[1] = 2;
numbers[2] = 3;
numbers[3] = 4;
numbers[4] = 5;

for (int i = 0; i < numbers.Length; i++)
{
    Console.WriteLine(numbers[i]);
}
```

В этом примере мы используем цикл `for` для вывода всех элементов массива на экран.

Цикл `foreach` позволяет пройтись по каждому элементу массива без необходимости знать его размер. Пример использования цикла `foreach` в C#:
```csharp
int[] numbers = new int[5];
numbers[0] = 1;
numbers[1] = 2;
numbers[2] = 3;
numbers[3] = 4;
numbers[4] = 5;

foreach (int number in numbers)
{
    Console.WriteLine(number);
}
```

В этом примере мы используем цикл `foreach` для вывода всех элементов массива на экран.

## Заключение

Массивы - это важный и часто используемый тип данных в программировании. Они позволяют хранить и обрабатывать большие количества данных, а также упрощают работу с ними.
